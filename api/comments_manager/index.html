    Constructor

    class CommentsManager(link, count, cookies)

      Manages the retrieval of comments for a YouTube video.

    Parameters:

      link str The URL of the YouTube video.
      count int The number of comments (raw preload)
      cookies list List of dicts.

    Properties

    							count
							int

      Amount of comments received during the first request. May differ from the actual number of comments after parsing.

    Methods

    							len()
							int

      Alias for count. Returns the number of comments. 
      len(yt.comments)

    							async get() ->
							list

      Asynchronously retrieves the comments for the YouTube video. Returns a list of Comment objects.

    Example

    import MyTube
import asyncio

link = "https://www.youtube.com/watch?v=l1OtC0q70g4"
yt = MyTube.YouTube(link)

async def main():
	comments = await yt.comments.get()
	for comment in comments:
		print(comment.author.name)
		print(comment.text)

asyncio.run(main())
