    Constructor

    class Subtitles(lang_code, lang_name, ext, url)

      Represents a subtitle file for a video.

    Parameters:

      lang_code str The language code. ('en', 'es')
      lang_name str The name of the language. ('English' or 'Spanish')
      ext str The file extension of the subtitle. ('vtt')
      url str The URL of the subtitle file.

    Properties

    							code
							str

      The ISO language code.

    							name
							str

      The humanized name of the language.

    							extension
							str

      The file extension of the subtitle file.

    							url
							str

      The URL for the subtitle file.

    Example

    pip install prettytable
    import MyTube
from prettytable import PrettyTable

yt = MyTube.YouTube("https://www.youtube.com/watch?v=Xj0Jtjg3lHQ")
t = PrettyTable(['Code', 'Name', 'Extension', 'URL'])

for sub in yt.subtitles:
	t.add_row([sub.code, sub.name, sub.extension, sub.url])
print(t)
